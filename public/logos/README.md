# Logos

Coloca aquí los archivos de logos:

- **Logo de Versu**: `versu-logo.svg` (o el formato que tengas: .png, .jpg, etc.)
- **Logos de Partners**: Coloca los logos de los partners en esta carpeta o en una subcarpeta `partners/`

Los archivos se referenciarán como:
- `/logos/versu-logo.svg` para el logo de Versu
- `/logos/partners/meta-logo.svg` para logos de partners (si los pones en subcarpeta)

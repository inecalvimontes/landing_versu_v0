# Assets Públicos

Esta carpeta contiene los archivos estáticos del proyecto.

## Estructura recomendada:

```
public/
  logos/
    versu-logo.svg (o .png, .jpg)
    partners/
      meta-logo.svg
      shopify-logo.svg
      [otros logos de partners]
```

Los archivos en esta carpeta se pueden referenciar desde los componentes usando rutas que empiezan con `/`.

Ejemplo: `/logos/versu-logo.svg`
